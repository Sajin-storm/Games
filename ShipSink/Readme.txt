Just double click the SHS.bat file to run the game in cmd

Game.java is the source code

Ship sink game
where the system generates 3 ships which is of 3 box length 
is placed randomly in a 7x7 grid.

You as a user has to guess where those 3 ships may be placed by the system.

the Y axis A to G and X axis 0 to 6
so user input can be a combination of Y axis and X axis eg(A0)

Try to guess the 3 ships in less number of moves as possible.

Have fun playing!!!
